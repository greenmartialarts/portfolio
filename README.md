# GreenMartialArts Portfolio

A high-performance, responsive portfolio website built with vanilla HTML, CSS, and JavaScript. Designed with a custom "Neon Green" aesthetic, it features advanced UI interactions like glassmorphism, 3D tilt effects, and real-time GitHub statistics.

![Project Preview](https://via.placeholder.com/800x400?text=Portfolio+Preview)

## Features

- **Premium Neon Design**: A striking dark theme with vibrant `#00ff41` accents and glassmorphism elements.
- **Interactive UI**:
  - **Custom Cursor**: A glowing cursor with a magnetic trailing ring (hides when mouse leaves window).
  - **3D Tilt Cards**: Project and stat cards respond to mouse movement with a tangible 3D effect.
  - **Scroll Reveal**: Sections elegantly fade in and slide up as you scroll.
- **Dynamic GitHub Stats**: A custom dashboard that directly fetches your data (Total Stars, Repos, Years Active, TS Projects) from the GitHub API without relying on third-party widgets.
- **Fully Responsive**: Optimized for all devices with a smooth mobile navigation menu.
- **Lightweight**: Built without heavy frameworks to ensure maximum load speed and performance.

## Tech Stack

- **HTML5**: Semantic markup for accessibility and SEO.
- **CSS3**: Custom variables, Flexbox/Grid layouts, and CSS animations.
- **JavaScript (ES6+)**: Custom logic for API fetching, cursor engine, and intersection observers.
- **FontAwesome**: Scalable vector icons.

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/greenmartialarts/portfolio.git
    ```
2.  **Open the project**:
    Simply open `index.html` in your web browser. No build step required!

## Deployment (GitHub Pages)

This project is ready for **GitHub Pages**.

1.  Push the code to your GitHub repository (e.g., `portfolio`).
2.  Go to **Settings** > **Pages**.
3.  Under **Source**, select `main` branch.
4.  Your site will be live at `https://greenmartialarts.github.io/portfolio`.
